Get started with Virtual Studio

  • 建立NG專案
  • 移動必要的檔案進入MVC專案
    • Src資料夾
    • package.json
    • angular.json
    • tsconfig.json
  • 還原package.json套件封裝
    • 在package.json上右鍵點選還原封裝
  • 更新tsconfig.json設定(Typescript編譯器的設定檔)
    • 新增include: [ “./src” ]
  • 更新angular.json設定(ng build的輸出路徑)
    • 修正outputPath: “./Scripts/libs”
  • 建置build版本
    • ng build
    • 輸出檔案會放在Scripts/libs，將該路徑加入ASP.NET MVC專案中
  • 更新ASP.NET View
    • 引入js
      • Scripts/libs/runtime.js
      • Scripts/libs/polyfills.js
      • Scripts/libs/styles.js
      • Scripts/libs/vendor.js
      • Scripts/libs/main.js
    • 加入html root
      • <app-root></app-root>
